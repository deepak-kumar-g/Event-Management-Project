# Event Management Project

A simple and responsive events webpage built using **HTML**.  
This page displays upcoming events with images, descriptions, timings, venue details, and contact information.

## Description

This repository contains a small static web page (`event.html`) and an `img/` folder with images used by the page. It's intended as a lightweight demo/event landing page that can be opened directly in a browser or served from a simple static server.
